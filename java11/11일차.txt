객체가 생성될 때
멤버변수값을 대입하면 됨.
=> 객체생성될 때 자동으로 호출되는 메소드
=> 객체생성될 때 자동으로 처리해야하는 부분
=> 객체생성될 때 자동으로 호출된다고 해서 "생성자"
=> 생성자 = constructor
=> public 클래스이름()
=> 생성자 메소드이름 클래스이름과 동일해야한다.
=> 생성자 메소드이름은 대문자로 시작!
=> 생성자 메소드는 객체생성시 자동 호출!!
=> 생성자 메소드는 여러개 만들 수 있다. (ok)
=> 생성자 오버로딩(다형성)
=> 생성자가 하나도 없는 경우
     파라메터 없는 기본 생성자가 자동으로 만들어진다.
=> 파라메터 없는 생성자를 "기본 생성자"라고 부른다.
